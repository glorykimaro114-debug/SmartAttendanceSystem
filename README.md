## Smart Attendance System
## OverView
Smart attendance System is a web application developed using django to help institutions manage employees attendance 
## Features  
-- Employees registration 
-- User login 
-- Attendance recording
-- Attendance report -- Admin dashboard

## Technologies Used :
Python , Django , HTML ,CSS , and SQLite.
## Installation

1. Clone the repository

bash
git clone https://github.com/glorykimaro114-debug/SmartAttendanceSystem.git


2. Navigate to the project directory

bash
cd SmartAttendanceSystem.


3. Install dependencies

bash
pip install -r requirements.txt

4. Apply migrations

bash
python manage.py migrate


5. Run the server

bash
python manage.py runserver
